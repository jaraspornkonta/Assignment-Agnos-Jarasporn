# Assignment-Agnos-Jarasporn

## Project Overview
This project contains automated test scripts developed using Robot Framework and SeleniumLibrary to test the Agnos web application.

## Test Coverage
The automation test cases include:

- Login Successfully
- Login With Invalid Password
- Search Patient By Name
- Search Patient With No Result
- Logout Successfully

## Tools Used
- Robot Framework
- SeleniumLibrary
- Python
- Chrome Browser

## How to Run the Tests

1. Install Python
2. Install Robot Framework
3. Run the test
4. Test report will be generated automatically:
- report.html
- log.html
